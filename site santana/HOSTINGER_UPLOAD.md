# Upload para Hostinger

1. Faça upload do arquivo `cidade-ja-news-theme.zip` pelo painel do WordPress em Aparência > Temas > Adicionar novo > Fazer upload do tema.
2. Se preferir via FTP, extraia a pasta `cidade-ja-news` dentro de `wp-content/themes/` no diretório do WordPress.
3. Ative o tema "Cidade Já News" em Aparência > Temas.
4. Para usar o Elementor, instale o plugin Elementor e ajuste os blocos na home.

Observação: o tema atual é uma estrutura básica, preparada para ser expandida com o layout editorial do site.
